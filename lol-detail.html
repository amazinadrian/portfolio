League of Legends Wiki

HTML, Javascript, CSS

This was my very first project that I created with HTML, Javascript and CSS. I was inspired for my love of the game "League of Legends". I'm a gamer and it was only fitting that I started a project with something I enjoy. With this project I was able to connect and fetch data via Riot's API of League of Legends. Users are able to search up a Champion by name. Once searched, an image will show with correlating data of statistics related to the Champion. This defintely was a fun first project where I was introduced to API's and grabbing that data to show visually on the frontend side.
